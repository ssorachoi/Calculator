description of each functions

Type of Function Description Example
Addition "+", used to get the sum of 2 or more number 2+2+1= 5
Subtraction "-", used to get the difference of 2 or more number 5-2-1= 2
Multiplication "*", used to get the product of 2 or more number 2*2*1= 4
Division "/", used to get the quotient of 2 or more number 4/2/1= 2
Modulo "%", used to get the remainder of 2 or more number 5%2%1= 0
